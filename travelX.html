About

Year Developed: 2023

I designed and developed a comprehensive multiple-page travel website from scratch, using my knowledge of HTML, CSS, and JavaScript. By integrating multiple APIs, I ensured accurate and up-to-date weather and travel information for various locations. Additionally, I incorporated a feature for AI-generated responses to provide users with personalized recommendations and insights.

Insights Gained: As a key challenge, I acquired new skills in working with APIs, focusing on ensuring proper data retrieval and display. The implementation of AI-generated responses required learning advanced techniques for integrating and managing AI services. I utilized Visual Studio Code as my primary development tool, leveraging its powerful features to optimize the code structure and enhance efficiency. This project not only showcased my technical proficiency but also demonstrated my ability to adapt and learn independently, solidifying my foundation in web development.
